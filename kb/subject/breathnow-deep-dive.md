## User Feedback Themes

### Positive
- "My BP went from high to normal in a couple of months without medication"
- 3-minute lower BP feature "works every time"
- Users report using 2-3 times daily
- Good Apple Watch integration

### Negative
- Pro membership "just isn't all that great" — perceived value gap
- Video playback issues reported
- Some users find the subscription tiers confusing (too many options)

## Strengths
1. Only app combining tracking + intervention
2. Strong scientific positioning (Cambridge VO2Max test, cited research)
3. Broad exercise library (breathing, yoga, qigong, meditation)
4. AI Health Coach adds personalisation
5. Active content marketing (Medium, YouTube, TikTok)
6. Sole focus on BP (not diluted across wellness)

## Weaknesses
1. English only — limits market
2. Likely solo developer — bus factor of 1
3. Wix website (not professional/custom)
4. Confusing subscription tiers (6+ options)
5. Pro value perception issues
6. No web version (app only)
7. 95.9 MB is hefty for a utility app
8. Requires iOS 17.6+ (cuts off older devices)
