# Sessions

## 2026-04-07 — Initial MVP Build

**What:** Built the complete MVP from scratch after researching BreathNow (Cardio Calm Ltd) as inspiration.

**Built:**
- Flask web app, mobile-first dark theme UI (Tailwind CDN)
- BP logging with AHA classification (Normal/Elevated/High Stage 1/High Stage 2)
- 5 breathing exercises: 4-7-8, Box, Slow BP, Coherent, IMST
- Animated breathing timer (SVG progress ring + scaling circle)
- 7-day trend chart (Chart.js)
- Session auto-logging
- Full history view
- Bottom nav (app-like mobile UX)

**KB seeded:**
- `kb/subject/`: breathing science (cited), hypertension stats (WHO/AHA/BHF), competitor analysis (8 apps), BreathNow deep dive
- `kb/project/`: architecture decisions, current state
